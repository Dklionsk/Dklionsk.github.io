Modern Art (Solution)
Each QR code decodes to another clue. All of these clues are entries in an ordered sequence. 

2: KHAAAAN! (from Star Trek II: The Wrath of Khan)
4: April (4th month)
5: Makalu (5th tallest mountain)
6: Half-Blood (Harry Potter book 6)
8: Van Buren (8th president)
9: Eccleston (9th Doctor Who)
The flavor text hints that you have to take the art apart and put it back together. Each number refers to one ninth of the code. Take those ninths and assemble them into a new code. That code decodes to the answer word, REVERIES. The metacode can be modified to give any solution word.
